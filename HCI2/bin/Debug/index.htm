Home tab

The initial window contains 6 tabs. The initial tab is for making schedule. On that tab you can add subject in selected classroom.

Classroom tab

Classroom tab is for adding new classroom, changing classroom or deleting classroom. There is a display of all classrooms.

Course tab

Course tab is for adding new course, changing course or deleting course. There is a display of all courses.

Subject tab

Subject tab is for adding new subject, changing subject or deleting subject. There is a display of all subjects.

Software tab

Software tab is for adding new software, changing software or deleting software. There is a display of all softwares.

Help tab
